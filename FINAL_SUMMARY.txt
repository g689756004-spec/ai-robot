🚀 GET STARTED IN 5 MINUTES
═══════════════════════════════════════════════════════════════════════════════

1. BACKEND (2 minutes)
   
   cd E:\React\robot
   pip install -r requirements.txt
   set ANTHROPIC_API_KEY=your_claude_key
   python main.py
   
   ✓ Backend running on http://localhost:8000

2. ANDROID (2 minutes)

   • Open Android Studio
   • File → Open → E:\React\robot
   • Build project (./gradlew build)
   • Deploy to device (./gradlew installDebug)
   
   ✓ App installed and ready
